# Eq2Graph

**Eq2Graph | React.js, Node.js, Express.js, Google Gemini AI**

•	Developed an AI-powered mathematical tool that converts hand-drawn images, typed equations, and word problems into interactive visual graphs.
•	Built a backend REST API using Node.js and Express to process multimodal inputs and leverage Google Gemini AI for accurate equation extraction. 
•	Designed a responsive UI with React and TailwindCSS, featuring real-time dynamic graphing and an intuitive canvas for user interactions.

## How to Run Locally

### Backend
1. Navigate to the `backend` folder: `cd backend`
2. Install dependencies: `npm install`
3. Create a `.env` file and add your Google Gemini API Key: `GEMINI_API_KEY=your_key_here`
4. Start the server: `npm run dev`

### Frontend
1. Navigate to the `frontend` folder: `cd frontend`
2. Install dependencies: `npm install`
3. Start the Vite dev server: `npm run dev`
